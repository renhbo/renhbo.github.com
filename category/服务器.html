新购VPS基本检测步骤

2011-08-16 第33周

作者: d0gate
类别: 服务器.

标签: vps

VPS 性能检测

当新够买一个VPS后，一般要检测机器的基本属性。本文以安装debian 6的VPS为例说明常用的步骤。

注: debian和ubuntu相比需要的内存更少，在低端VPS上建议用debian.

查看IP:

/sbin/ifconfig

查看发行版:

uname -a

磁盘读写性能:

dd if=/dev/zero of=test bs=64k count=4k oflag=dsync

以树状查看当前进程:

pstree

查看某一个进程内存占用情况:

ps -C dropbear v

查看内存占用:

free

更新为最快的发布源, 将默认的us改为离你最近的源:

vi /etc/apt/sources.list

去掉不用的包:

$su
#dpkg -l
#aptitude remove your_package

更新系统:

#aptitude update
#aptitude upgrade

系统软件优化

  • ssh->dropbear:

    aptitude install dropbear
vi /etc/default/dropbear
NO_START=0 (设置成为自动启动)

  • bash -> pdksh:

    aptitude install pdksh
使用chsh改变默认的shell(/bin/ppdksh)

添加用户:

#cat /etc/default/useradd
#adduser abc

更多内容参见 VPS新手指南 。
