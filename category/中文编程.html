我的第一个中文程序

2011-08-27 第34周

作者: d0gate
类别: 中文编程.

打开编辑器，输入下面的内容:

打印("中文编程真有趣！")

注意: ()和""都是半角，不是（）“”。

运行的结果是:

中文编程真有趣！

打印( ) 是Python的关键字，把内容打印在屏幕上。这是一个函数，需要把打印的内容放在 ( ) 里。 "中文编程真有趣！" 是字符串，字符串用一对双引号"或者单引号'来表示，如"中文"和'中文'。用双引号或者单引号没有什么区别。

    中文编程一直是一个热门话题，也有很多种尝试，在 Python 环境中， 汉蟒 和 周蟒 (zhpy)是致力于中文编程的软件。汉蟒是基于python2.3源代码修改，增加了中文关键字支持。周蟒通过解释程序预处理翻译关键字，更具有灵活性。

    本文将尝试使用中文编程来讲 Python3 入门。由于本人水平有限，欢迎提出批评指正。
