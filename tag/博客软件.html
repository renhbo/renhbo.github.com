1   Pelican简介

Pelican 是一款博客软件，文章内容以纯文本格式写作，保存在本地，通过命令自动生成页面和索引文件。生成页面文件后在手工向Web服务器发布。

与著名的 Wordpress 不同，用户不需要安装 mysql 数据库，不需要 PHP ，是小内存VPS博客的首选。 另一个好处是所有的文章都是静态页面，访问速度飞快。

Pelican的名字来源于法语Celipan，是笔记本的意思.

2   Pelican特点

  • 一个博客软件
  • 使用 rst 或者 markdown 格式写作
  • 集成外部的 disqus 评论系统（可选）
  • 支持主题，模板是jinja2
  • 同一文章多语种发布
  • 生成RSS/Atom订阅
  • 导入wordpress/dotclear文章或者其他RSS格式
  • 集成第三方工具twitter/google analytics（可选）
  • 开放源码的 Python 程序
  • 支持Linux和Windows

3   环境准备

  • Python2.7
  • Ubuntu Linux

4   命令行下安装

$ sudo easy_install pelican

5   使用

5.1   使用快捷向导建立博客

$ pelican-quickstart
Welcome to pelican-quickstart v2.7.2.

This script will help you creating a new Pelican based website.

Please answer the following questions so this script can generate
the files needed by Pelican.

5.2   生成页面

pelican src -o output -s pelican.conf.py

5.3   使用make发布

$ make
Makefile for a pelican Web site

Usage:
   make html                        (re)generate the web site #(重)生成站点内容
   make clean                       remove the generated files #清空生成的文件。
   ftp_upload                       upload the web site using FTP #通过FTP发布站点
   ssh_upload                       upload the web site using SSH #通过SSH发布站点
   dropbox_upload                   upload the web site using Dropbox #通过dropbox发布站点

5.4   修改配置文件 pelican.conf.py

$ vi pelican.conf.py
# static paths will be copied under the same name
STATIC_PATHS = ["pictures",]

# A list of files to copy from the source to the destination
FILES_TO_COPY = (('extra/robots.txt', 'robots.txt'),)


WITH_PAGINATION = True
DEFAULT_PAGINATION = 7

5.5   生成页面

$ make clean; make html

5.6   查看结果

生成的页面在output目录中。

cd output
python -m SimpleHTTPServer 8000

5.7   发布

页面发布可以通过FTP等工具上载到服务器页面。
